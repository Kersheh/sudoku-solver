# sudoku-solver

Author: Matthew Breckon

A heuristic approach at solving sudoku puzzles.

1. Compile
> $ make

2. Usage
> $ ./sudoku sudoku_puzzle.txt

3. Puzzle Format:

  0 9 0 8 7 0 3 0 0 
  
  4 0 1 0 0 2 0 8 0 
  
  7 0 3 4 0 0 0 0 0 
  
  3 0 8 0 0 0 0 7 1 
  
  6 0 0 0 0 0 0 0 9
  
  5 7 0 0 0 0 2 0 3 
  
  0 0 0 0 0 9 7 0 8 
  
  0 0 2 0 1 0 4 0 5
  
  0 0 7 0 4 8 0 3 0
